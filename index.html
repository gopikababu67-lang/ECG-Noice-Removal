// Signal Flow

Processing Pipeline

From raw noisy ECG to clean clinical-grade waveform in four stages.

01 📥
Acquisition

12-lead ECG sampled at 250–500 Hz. Pre-amplification & ADC conversion. Accepts MIT-BIH, EDF, CSV formats.

02 🔍
Noise Detection

CNN classifier identifies noise type & severity across frequency bands: BW (0–1 Hz), PLI (50 Hz), MA (100+ Hz).

03 🧠
DL Denoising

1D-UNet autoencoder with skip connections reconstructs clean signal while preserving QRS morphology and T-wave amplitude.

04 📊
Output & Analysis

SNR, PRD, RMSE quality metrics. R-peak detection, HRV analysis, arrhythmia screening on cleaned signal.

// Model Architecture

Deep Learning Stack

A hybrid architecture combining convolutional autoencoders with attention mechanisms for adaptive noise suppression.

📐
1D-UNet Autoencoder

Encoder–decoder with skip connections at each depth level. Preserves fine QRS morphology while removing broadband noise. Trained on MIT-BIH + PhysioNet datasets.

Core Model
🎯
Attention Gate

Self-attention mechanisms focus the decoder on clinically significant features — P, QRS, T waves — suppressing irrelevant frequency components adaptively.

Attention Layer
🔊
Noise Classifier CNN

Pre-network CNN analyses 2-second windows to classify noise type and severity. Routes signal to specialized denoising branches for optimal performance.

Classifier
⚡
LSTM Temporal Context

Bidirectional LSTM layers capture long-range temporal dependencies across heartbeat cycles — critical for removing quasi-periodic baseline wander artifacts.

Temporal
📉
Adaptive Filtering

Post-DL adaptive Wiener filter cleans residual powerline interference (50/60 Hz) without distorting adjacent frequency bands.

Post-processing
🏆
Training Dataset

Trained on 48 MIT-BIH records + 10,000 synthetic noisy pairs. Tested on PhysioNet Challenge 2017. Augmented with clinical noise profiles from PTB-XL.

Dataset
// Handled Artifacts

Noise Types Removed

Our model is trained to remove four clinically significant noise categories that affect ECG diagnostic quality.

Baseline Wander

Low-frequency drift (0.05–0.5 Hz) caused by respiration and electrode motion.

85% removal
Powerline Interference

50/60 Hz sinusoidal noise from electrical equipment coupling.

99% removal
Muscle Artifact

High-frequency EMG contamination from skeletal muscle contractions.

78% removal
Motion Artifact

Broadband transient noise from patient movement and electrode impedance changes.

72% removal
